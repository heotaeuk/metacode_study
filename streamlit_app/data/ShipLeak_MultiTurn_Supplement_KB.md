## KB-13. 누설 위치 표시 및 마킹 절차

| 항목 | 내용 |
|---|---|
| Category | Leak Location Marking / Bubble Test Follow-up |
| 대표 질문 | 누설 위치 표시는 어떻게 하나요? / 분필로 하나요? / 다른 무엇을 하면 되나요? / 기포 발생 위치는 어떻게 기록하나요? |
| 한글 검색 키워드 | 누설 위치 표시, 누설 부위 마킹, 분필 표시, 마커 표시, 태그 부착, 사진 기록, 기포 위치 기록 |
| 영문 검색 키워드 | Leak location marking, Chalk mark, Paint marker, Tagging, Photo record, Bubble location |

### 목적
누설 위치 표시는 정비자가 실제 누설 부위를 놓치지 않고 후속 조치, 재시험, 사진 기록을 연결하기 위한 절차이다. 누설 위치는 비눗물 테스트, Leak Test, 압력 유지 시험, 가스 검지 결과를 바탕으로 표시한다.

### 사용 가능한 표시 방법
1. **분필**  
   - 고온이 아니고 표면이 건조한 경우 임시 표시용으로 사용할 수 있다.
   - 젖은 표면, 기름진 표면, 고온 표면에서는 지워질 수 있으므로 보조 수단으로 사용한다.

2. **페인트 마커 또는 산업용 마킹펜**  
   - 금속 표면에 비교적 명확하게 남길 수 있어 현장 표시용으로 적합하다.
   - 장시간 보존이 필요한 경우 분필보다 적합하다.

3. **내유성/내수성 태그**  
   - 밸브, 플랜지, 피팅 근처에 묶어서 누설 위치를 식별한다.
   - 표면에 직접 표시하기 어려운 경우 사용한다.

4. **테이프 또는 라벨**  
   - 표면 오염이 적고 접착 가능한 경우 임시 위치 표시로 사용한다.

5. **사진 기록**  
   - 누설 부위, 기포 발생 위치, 압력계, 밸브 ID가 함께 보이도록 촬영한다.
   - 표시가 지워질 수 있으므로 사진 기록을 반드시 병행한다.

6. **도면 또는 점검표 기록**  
   - Valve ID, 위치, 누설 방향, 누설 부위 번호를 기록한다.

### 권장 절차
1. 시험 전 유체와 압력 조건을 확인한다.
2. 안전한 시험 매체와 비눗물 용액을 준비한다.
3. 의심 부위에 비눗물 용액을 도포한다.
4. 기포가 발생하는 정확한 위치를 확인한다.
5. 기포 발생 위치와 발생 시간을 기록한다.
6. 압력을 낮추거나 안전 상태를 확보한 뒤 누설 위치를 표시한다.
7. 표시 시에는 분필, 마킹펜, 태그, 사진 기록 중 현장 조건에 맞는 방법을 선택한다.
8. 누설 위치를 표시한 뒤 체결부, Seal, Gasket, Seat, Disc, Packing 등 원인 부위를 확인한다.
9. 조치 후 동일 조건에서 재시험하여 기포가 소멸했는지 확인한다.

### 주의사항
- 가연성 가스 또는 위험 유체가 의심되면 먼저 접근 통제, 점화원 제거, 환기, 가스 검지를 수행한다.
- 누설 중인 상태에서 무리하게 가까이 접근하지 않는다.
- 고온 표면, 회전체, 전기 설비 주변에서는 일반 테이프나 분필 사용이 제한될 수 있다.
- 표시만 하고 끝내지 말고 반드시 사진, Valve ID, 위치, 시험 압력, 시간, 조치 내용을 기록한다.
